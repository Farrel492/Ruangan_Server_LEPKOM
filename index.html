Sistem Pendingin Ruangan
Ruangan server menggunakan konsep hot aisle / cold aisle containment — rak server diatur berhadap-hadapan sehingga udara dingin masuk dari depan rak (cold aisle) dan udara panas dibuang ke belakang rak (hot aisle). AC/CRAC unit mensuplai udara dingin ke cold aisle dan mengambil udara panas dari hot aisle untuk didinginkan kembali.

Physical Security System
🔐
Kartu Akses (RFID)
Active
Pintu server room dilengkapi reader kartu RFID. Hanya kartu yang terdaftar bisa membuka pintu. Log akses tersimpan otomatis — siapa masuk, jam berapa. Teknisi LEPKOM wajib tap kartu setiap kali masuk/keluar.
✓ Pintu terkunci · Last entry: 08:04 · Sulthan
📹
CCTV
Recording
Kamera CCTV di sudut ruangan dan di depan rak server. Rekaman disimpan 30 hari. Terhubung ke monitoring room. Resolusi 1080p dengan night vision.
✓ 3 kamera aktif · Recording 24/7
🚨
Smoke & Fire Detector
Normal
Sensor asap ionisasi terpasang di plafon. Terhubung ke sistem alarm gedung. Jika terdeteksi asap, alarm berbunyi dan pintu darurat terbuka otomatis. Dilengkapi FM-200 suppression system (bukan air — agar tidak merusak server).
✓ No smoke detected · Last test: 3 hari lalu
💧
Water Leak Sensor
Dry
Sensor kebocoran air di bawah lantai raised floor dan dekat AC unit. AC yang bocor adalah penyebab kerusakan server yang sering terjadi. Alert langsung dikirim ke HP teknisi jika terdeteksi.
✓ Dry · 4 sensor aktif
⚡
Anti-Static & Grounding
OK
Raised floor anti-static untuk cegah ESD (Electrostatic Discharge) yang bisa merusak komponen. Teknisi wajib pakai wrist strap anti-static saat pegang hardware. Rak server terhubung ke grounding system gedung.
✓ Grounding resistance: 0.8Ω (normal <1Ω)
🌡️
Environment Monitoring
Monitoring
Sensor suhu dan kelembaban di setiap rak dan corner ruangan. Alert otomatis via email/SMS ke admin jika suhu >27°C atau humidity >60%. Data disimpan untuk analisis tren jangka panjang.
⚠️ Rack B hot aisle 29.1°C — monitor lebih ketat
